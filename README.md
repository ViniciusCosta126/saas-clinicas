# Saas - Atendimentos de clinicas

#### Tecnologias utilizadas

**Backend:** Laravel

**Frontend:** Blade laravel,scss

**Banco de dados:** MySql,redis,mongo


#### Objetivo inicial:
Criar um sistema (saas) para gerenciamento de atendimentos de clinicas de psicologia. A ideia inicial do projeto vai servir apenas para clinicas mas depois talvez seja expandido para mais coisas
